Solution 2: Run a Local Web Server

Option A: Python (if installed)

# Navigate to your project folder, then run:
python -m http.server 8000

# Then open: http://localhost:8000/register.html

Option B: Node.js (if installed)

# Install http-server globally:
npm install -g http-server

# Navigate to your project folder, then run:
http-server -p 8000

# Then open: http://localhost:8000/register.html

Option C: VS Code Live Server Extension

  1. Install the "Live Server" extension in VS Code
  2. Right-click on register.html
  3. Select "Open with Live Server"

Why This Happens

When you open HTML files directly (file://), browsers block requests to external APIs for security reasons. You need to serve your files through HTTP (http:// or https://) for the fetch requests to work properly.
